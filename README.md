# CRC Kiosk Version A 2.1 Datafix

This version includes the content-locked community education kiosk with anonymous local engagement tracking.

## How to run
Open `index.html` in a browser.

## Engagement data
Engagement data begin when a user taps **Start Learning**. Data are stored locally in the browser's `localStorage` on that device only.

Admin export saves any current active session and downloads a CSV with anonymous session-level fields only.
